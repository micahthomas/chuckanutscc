# Chuckanut Sports Car Club website

Rebuild of [chuckanutscc.org](https://www.chuckanutscc.org) — a calendar-and-content site for the Bellingham, WA autocross club, founded 1956.

## Stack

- **[Astro 6](https://astro.build/)** (server output) + React islands for interactive bits
- **Cloudflare Workers** (Static Assets adapter) — hosting
- **Cloudflare D1** (SQLite) — every persisted record
- **Cloudflare R2** — photos, course maps, venue images
- **Cloudflare Access** — gates `/admin/*` (free for ≤50 officers)
- **Stripe Checkout** — annual membership payments (offline mock for local dev)
- **Resend** — event-reminder emails (offline log-only for local dev)
- **Tailwind v4** — styling, all in one stylesheet
- **Leaflet** — venue maps with ESRI satellite imagery
- **TipTap-style markdown editor** (custom React island) — admin authoring
- **OpenStreetMap** + **Google Maps** for directions
- **Scorekeeper** (wwscc.org) — registration and results, integrated via their public API

Total monthly cost at the club's scale: **$0** for hosting/DB/storage, plus domain renewal and Stripe's per-transaction fee.

## Local development

```sh
pnpm install
pnpm demo                    # resets local D1, seeds, starts dev at :4321
```

`pnpm demo` boots a fully-populated site with two events' worth of photos, every page, two past seasons of events from Scorekeeper, the real chuckanutscc.org shop catalog, and demo members. The admin at `/admin` is unguarded locally — you log in as `dev@localhost` automatically.

### Other useful commands

```sh
pnpm dev                     # just start the server (no reseed)
pnpm check                   # TypeScript / Astro type-check
pnpm build                   # build the Worker bundle
pnpm db:reset:local          # wipe local D1 and re-run migrations
pnpm db:seed:local           # apply demo.sql + every seeds/imports/*.sql

# Importers (write SQL to seeds/imports/, push files to R2)
bash scripts/import-wix-images.sh
node scripts/import-photos.mjs --src DIR --event ID --photographer ID --folder-url URL
node scripts/import-course-map.mjs --file IMG --event ID
node scripts/import-scorekeeper-season.mjs --series cscc2026 --season season_2026

# End-to-end smoke test (Playwright + system Chromium)
node scripts/verify.mjs
```

## Deploying

See [DEPLOY.md](./DEPLOY.md) for a step-by-step walkthrough. Short version:

```sh
pnpm wrangler login
pnpm wrangler d1 create cscc          # paste id into wrangler.jsonc
pnpm wrangler r2 bucket create cscc-media
pnpm db:migrate:remote
pnpm db:seed:remote
pnpm media:push:remote
pnpm deploy
```

## Project structure

```
src/
  layouts/        PublicLayout + AdminLayout
  pages/          Astro routes
    admin/        Admin UI (Cloudflare Access protected in prod)
      api/          Form-post handlers
    api/          Public form handlers (contact, membership, merch request, …)
    events/       Public event pages
    shop/         Merch (request-only, no Stripe in v1)
    ...
  components/
    public/       PhotoMosaic, EventCard, SatelliteMap
    admin/        ImageUpload, MarkdownEditor, EventForm, …
  lib/
    db.ts         D1/R2 binding accessors via cloudflare:workers
    queries.ts    All read queries — kept in one file on purpose
    mutations.ts  Membership payment + reminder upsert helpers
    scorekeeper.ts  Live event picker for the admin form
    maps.ts       OSM iframe + Google Maps directions URL builders
    markdown.ts   Server-side marked renderer
    stripe.ts     Checkout — auto-falls back to offline mock when no key
    email.ts      Resend wrapper — same offline pattern
    drive.ts     Google Drive photo sync (offline simulator generates SVGs)
    reminders.ts  Event reminder cron logic
  middleware.ts   /admin guard (reads Cloudflare Access header)

migrations/       D1 SQL migrations (numbered, applied via wrangler)
seeds/
  demo.sql        Hand-authored demo data
  imports/        Generated by import scripts (gitignored)

scripts/          Import scripts, the seed runner, deployment helpers
```
